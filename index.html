    Unordered list

  • an unordered list starts with an ul tag
  • every list item is added with an li tag. This needs an opening and closing tag
  • the unordered list ends with an closing ul tag

    Ordered list

  1. an ordered list follows the same pattern as the unordered list
  2. it starts and ends with an ol tag
  3. and uses li tags to insert content into the list
